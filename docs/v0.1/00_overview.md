# GDGoC Japan Wiki — Product Overview (v0.1)

## Problem Statement

GDGoC (Google Developer Groups on Campus) chapters in Japan face a persistent knowledge-loss problem. Valuable institutional knowledge — event venues, speaker contacts, project tips, community guidelines — disappears when student members graduate or chapters become inactive. There is no shared, durable, structured knowledge base for the growing network of GDGoC Japan chapters.

## Vision

A bilingual (Japanese / English), AI-powered wiki platform that makes it effortless for GDGoC Japan chapter members to capture, organize, and share community knowledge — so that no insight is ever lost again.

## Goals (v0.1)

1. **Capture** — Accept raw inputs (text, images, Google Docs) and let AI structure them into wiki pages.
2. **Organize** — Present content in a Confluence-style hierarchical wiki.
3. **Share** — One shared wiki space accessible to all GDGoC Japan chapters.
4. **Translate** — Every page available in both Japanese and English; users switch languages freely.
5. **Control** — Role-based access via Google Sign-In.

## Non-Goals (v0.1)

- Per-chapter isolated spaces (considered for v0.2).
- Real-time collaborative editing (e.g., Google Docs-style simultaneous editing).
- Native mobile apps.
- Offline access.
- PDF / spreadsheet file ingestion (post-v0.1).

## Primary Users

| Role | Description |
|------|-------------|
| **Chapter Member** | Active member of any GDGoC Japan chapter. Can read all pages and create/edit pages they own. |
| **Chapter Lead** | Chapter organizer. Has elevated editing rights; can publish/unpublish pages and manage tags (global taxonomy only in v0.1). |
| **Admin** | Platform administrator. Full control: user management, page deletion, system settings. |
| **Cross-Chapter Viewer** | Authenticated member browsing another chapter's content. Read-only by default. |

## Success Metrics (v0.1)

- At least 3 GDGoC Japan chapters onboarded.
- Average time to create a structured wiki page from raw input < 2 minutes.
- Pages available in both languages for ≥ 90% of published content.
- Zero knowledge loss on graduation (measured by content retention survey).
